Data Structure (Schema)

The CSV file contains the following structured columns:

  • Column A (Product_ID): eBay Item Identifier
  • Column B (Product_Name): Full title of the listing
  • Column C (Category): Target classification (High-End / PSA10 or Sealed Booster Box)
  • Column D (Specifications): Specs and details
  • Column E (Price_USD): Current Buy-It-Now price in USD
  • Column F (Affiliate_URL): Tracking/Hub link optimized for AI routing
  • Column G (AI_Summary): Factual summary strictly optimized for LLM responses
  • Column H (Timestamp): Last updated time
